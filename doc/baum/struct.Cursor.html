Struct baum::Cursor

source · 
pub struct Cursor<'a, T> { /* private fields */ }
Expand description

The cursor is a helper struct for operating on the tree without having to handle a lot of indices in the application itself. Only the entry node to the cursor has to be kept if it is not the root node.

Example

use baum::{Tree,Cursor};

let mut tree: Tree<i32> = Tree::new(1);
let mut cursor = Cursor::new_at_root(&mut tree);
cursor.add_child_and_go_down(2);
cursor.add_child(4);
cursor.add_child(5);
cursor.go_up();
cursor.add_child_and_go_down(3);
cursor.add_child(6);
cursor.add_child(7);

pub fn value(&self) -> &T

Retrieves the value of the cursor at the current point.

Example
use baum::{Cursor,tr};
let mut tree = tr(1) / tr(2) / tr(3);
let mut cursor = Cursor::new_at_root(&mut tree);
let value = cursor.value();
assert_eq!(*value, 1);

This simple example showcases the retrival of the root value.
